JSON::Tiny

Minimal JSON in a no non-core dependency configuration.

DESCRIPTION

Lightweight, fast, relaxed JSON in a stand-alone module with no non-core dependencies.

INSTALLATION

To install this module, run the following commands:

perl Makefile.PL
make
make test
make install

DEPENDENCIES

There are no non-core dependencies. This module requires a minimum Perl version of 5.10.0.
